# Help Desk Home Lab

A beginner help desk home lab built with UTM and Windows 11 to practice common IT support tasks.

## Project Overview
This lab was created on a MacBook Pro using UTM and a Windows 11 virtual machine. The goal was to practice common entry-level IT support tasks in a home lab environment and document the work like a real help desk technician.

## Tools Used
- UTM
- Windows 11
- Local Users and Groups
- File Explorer
- Notepad

## Lab Setup
- Created a Windows 11 virtual machine in UTM
- Created an administrator account named `labadmin`
- Created a standard user account named `jdoe`
- Renamed the computer to `HELPDESK-LAB`

## Tickets Completed

### HD-001: Password Reset
- Reset local account password for user `jdoe`
- Verified successful login
- Documented resolution

### HD-002: App Access Support
- Located Calculator for user
- Confirmed application opened successfully
- Pinned app to Start/taskbar for easier access

### HD-003: Shared Folder Access
- Created shared folder for user file access
- Granted read/write access
- Verified file creation in shared folder

### HD-004: Printer Troubleshooting
- Opened printer settings
- Searched for available printers
- Reviewed manual add options
- Documented troubleshooting steps

### HD-005: Desktop Shortcut and File Organization
- Created a work folder structure
- Added Reports, Invoices, and Reference folders
- Created easier access for the user
- Verified file access

## Skills Practiced
- Windows user account management
- Password reset
- Basic file sharing
- Application support
- Printer troubleshooting
- File organization
- End-user support
- Ticket documentation
